Projects

Mav's Ice Cream Emporium

Developed a graphical user interface program for an ice cream parlor written under Linux in C++ and gtkmm. Features included tracking of outgoing and incoming transactions, balances, records of orders, customer information, employee information, and item inventory. Enhancements made included company logos, visual menus with a photo gallery and descriptions of items for sale. And multi-tier user access levels.

Mav's Bookstore

Created an online university bookstore using HTML5 and CSS. The website implemented a MySQL database with PHP functions. The database held a variety of items, descriptions, prices, customers, and order data. Features included customer information, item information, invoices, prices, and media types.
